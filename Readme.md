# Portfolio

Ceci est un Portfolio fait en pages Razor.
La plupart des informations présentées dans ce portfolio sont stockées dans une base de données.

## Prérequis

1. postgres
2. .net core 2.2

## .env

Variable du fichier .env à la racine du projet :

```
DB_HOST
DB_NAME
DB_USER
DB_PASSWORD
SENDGRID_API_KEY
ISALREADYSEED
```
